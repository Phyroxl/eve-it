# EVE iT Suite - Resumen Técnico de Arquitectura y Funcionalidad

Este documento sirve como contexto para que una IA comprenda rápidamente la estructura y el propósito de la suite EVE iT.

## 1. Propósito General
EVE iT es una suite de herramientas de asistencia para el juego EVE Online, diseñada para funcionar como un overlay no intrusivo. Sus tres pilares son:
1. **ISK Tracker:** Monitorización en tiempo real de ingresos mediante el parseo de logs.
2. **Chat Translator:** Traducción automática de canales de chat con soporte para Zkillboard.
3. **EVE Replicator Hub:** Sistema avanzado de duplicación de ventanas (multi-boxing) con captura de alto rendimiento.

## 2. Tecnologías Principales
* **Lenguaje:** Python 3.10+.
* **Interfaz Gráfica:** PySide6 (Qt para Python) con estilos "Premium Neon".
* **Captura:** Win32 API avanzada (GDI, User32, PrintWindow).
* **Comunicación:** Señales de Qt para interactividad entre componentes.

## 3. Mejoras Recientes (Replicator 2.0)
El sistema de replicación ha sido rediseñado para ofrecer una experiencia fluida y profesional:

### A. Motor de Captura Híbrido y Fluidez
* **Captura Inteligente:** Prioriza la captura directa del escritorio (GDI `StretchBlt`) cuando el juego está al frente para alcanzar FPS nativos, volviendo automáticamente a la captura interna si el juego se tapa.
* **Estabilidad Win32:** Limpieza estricta de descriptores (GDI Objects) para evitar fugas de memoria y crasheos en sesiones de larga duración.
* **Gestión de Foco:** Sistema de enfoque dinámico que evita que las réplicas "congelen" el renderizado de EVE Online al interactuar con ellas.

### B. UI/UX y Diseño Dinámico
* **Panel de Control Adaptativo:** Rediseño estilo "Tarjetas Neón" que ajusta su altura automáticamente según la cantidad de personajes detectados.
* **Interfaz Ultra-Limpia:** Eliminación de marcos internos y HUDs intrusivos. Todas las opciones de control se han trasladado al menú contextual (click derecho).
* **Asistente de 2 Pasos:** Simplificación radical del flujo de configuración, eliminando pasos redundantes.

### C. Controles de Precisión y Atajos
* **Dynamic Trimming (NUEVO):** 
    * `Ctrl + Scroll`: Recorta o amplía los bordes superior e inferior simultáneamente.
    * `Alt + Scroll`: Recorta o amplía los bordes laterales de forma simétrica.
* **Interacción Directa (Portal):** Cambio de modificador a `Alt + Click` para enviar acciones directas al juego a través de la réplica.
* **Re-selección en Caliente:** Opción `⚡ (Cambiar Zona)` que permite reajustar la región de captura instantáneamente desde el menú de la réplica.
* **Joystick Contextual:** Movimiento de la región con pasos ultra-finos (0.5%) mediante el menú derecho.

## 4. Estructura del Proyecto
* `/controller`: Lógica central, `TrayManager` y `ReplicatorHub`.
* `/overlay`: Motor de renderizado `ReplicationOverlay` y selector de región.
* `/translator`: Motores de lectura y traducción de logs.
* `/core`: Trackers de sesión y lógica de ISK.
* `update_github.bat`: Script de automatización para despliegue rápido.

## 5. Notas para Mantenimiento
* **Estética:** Mantener siempre el esquema de colores Cian (#00c8ff) y Verde Neón (#00ff9d).
* **Cierre de Recursos:** Al cerrar una réplica, es CRÍTICO desconectar señales y detener el hilo de captura para evitar accesos a memoria inválida.
* **Localización:** Usar `i18n.py` para todas las cadenas de texto nuevas.
