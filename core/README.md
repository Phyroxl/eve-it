# ⚙️ Core
El núcleo del sistema. Contiene la lógica base, orquestación de la aplicación y clases fundamentales que no pertenecen a un módulo específico.

## Archivos futuros (Propuestos):
- `app_controller.py`: Orquestador principal (actualmente en la raíz o `controller/`).
- `base_module.py`: Clase base para todos los módulos de la suite.
- `events.py`: Sistema de comunicación entre módulos.
